## 二、星梦动画是什么？

星梦动画是一款基于 **Vue 3 + TypeScript + Vite + Electron** 构建的桌面端 AI 画布工坊。

### 🎯 核心功能

**1. 节点式画布工作流**
- 可视化拖拽节点，搭建自己的创作工作流
- 支持文本、图片、视频等多种节点类型
- 灵活的节点连接和数据传递

**2. 多模态内容生成**
- 📝 **文本生成**：支持多种大语言模型
- 🖼️ **图片生成**：AI 画图，风格多样
- 🎬 **视频生成**：从文字或图片生成视频
- 🔗 **参考图联动**：支持多图参考和风格迁移

**3. 智能串联管理**
- 上下游节点内容自动传递
- 任务状态实时展示
- 生成结果预览与一键保存

**4. 多模型多渠道**
- 支持优尚、Flow2 等多个 API 渠道
- 灵活切换不同模型和供应商
- 满足各种创作场景需求

### 💡 如何使用？

使用星梦动画非常简单：

**步骤 1：下载安装**
- 从 GitHub Releases 下载最新版本
- 双击安装包，按照提示完成安装

**步骤 2：配置 API**
- 打开软件，进入设置页面
- 配置你的 API 地址和密钥
- 选择喜欢的模型和供应商

**步骤 3：开始创作**
- 在画布上拖拽添加节点
- 连接节点，搭建工作流
- 点击生成，等待作品出炉！

## 三、如何拷贝一份代码，生成自己的软件？

如果你是开发者，想要基于星梦动画打造自己的软件，非常简单！

### 🚀 快速开始

**1. Fork 项目**
- 访问 GitHub 仓库：https://github.com/dear850223/ai-video-generator
- 点击右上角的 "Fork" 按钮，将项目复制到你自己的账号下

**2. 克隆代码**
```bash
git clone https://github.com/你的用户名/ai-video-generator.git
cd ai-video-generator
```

**3. 安装依赖**
```bash
npm install
```

**4. 本地开发**
```bash
npm run dev          # 前端开发模式
npm run electron:dev # 完整开发模式
```

**5. 打包发布**
```bash
npm run build        # 完整构建
npm run electron:build # 打包桌面应用
```

打包完成后，安装包会输出到 `release/` 目录，你就可以分发给用户使用了！

### 🎨 自定义修改

你可以根据自己的需求自由修改：
- 修改界面风格和主题
- 添加新的功能节点
- 集成更多 AI 模型
- 调整打包配置

只要遵守 GPL-3.0 许可证，你可以尽情发挥创意！
