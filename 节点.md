| 节点类型 | 图标 | 描述 |
|---------|------|------|
| text | 📝 | 文本处理节点 |
| image | 🖼️ | 图片生成节点 |
| video | 🎬 | 视频生成节点 |

## 二、Vue Flow 官方示例中可以参考的节点类型

### 1. 数据输入/输出类

| 节点类型 | 参考来源 | 功能描述 |
|---------|---------|---------|
| ValueNode | Math 示例 | 数值输入框节点 |
| ColorSelectorNode | CustomNode 示例 | 颜色选择器节点 |
| RGBNode / RGBOutputNode | RGBFlow 示例 | RGB 颜色混合节点 |
| ResultNode | Math 示例 | 结果显示节点 |

### 2. 逻辑运算类

| 节点类型 | 参考来源 | 功能描述 |
|---------|---------|---------|
| OperatorNode | Math 示例 | 运算符节点（+ - * /） |
| ConditionNode | - | 条件判断节点（if/else） |
| SwitchNode | Switch 示例 | 分支切换节点 |

### 3. 流程控制类

| 节点类型 | 参考来源 | 功能描述 |
|---------|---------|---------|
| ProcessNode | Layout 示例 | 流程处理节点（带运行状态） |
| StartNode | - | 起始节点 |
| EndNode | - | 结束节点 |
| GroupNode | Nesting 示例 | 分组/容器节点 |

### 4. 数据处理类

| 节点类型 | 参考来源 | 功能描述 |
|---------|---------|---------|
| FilterNode | - | 数据过滤节点 |
| TransformNode | - | 数据转换节点 |
| MergeNode | - | 数据合并节点 |
| SplitNode | - | 数据拆分节点 |

### 5. 交互类

| 节点类型 | 参考来源 | 功能描述 |
|---------|---------|---------|
| ToolbarNode | NodeToolbar 示例 | 带工具栏的节点 |
| ResizableNode | NodeResizer 示例 | 可调整大小的节点 |
| TeleportableNode | Teleport 示例 | 可传送的节点 |

## 三、推荐优先实现的节点（适合 AI 创作项目）

基于 AI 创作画布的项目定位，建议优先实现以下节点：

| 优先级 | 节点类型 | 描述 |
|-------|---------|------|
| 🔴 高 | 音频节点 (audio) | 音频生成/处理，目前 icon 已预留 |
| 🔴 高 | 数据输入节点 (input) | 可配置参数的输入节点 |
| 🔴 高 | 输出展示节点 (output) | 显示最终结果的节点 |
| 🟡 中 | 开始/结束节点 | 流程的起点和终点 |
| 🟡 中 | 条件判断节点 | 条件分支控制 |
| 🟢 低 | 分组节点 | 用于组织画布，方便管理 |
