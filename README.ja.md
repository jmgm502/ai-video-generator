# AI Video Generator（星夢動画）

**Vue 3 + TypeScript + Vite + Electron** で構築されたデスクトップ AI キャンバスワークショップで、ノードベースのキャンバスを通じてテキスト、画像、動画などのマルチモーダルコンテンツの生成、連鎖、管理をサポートします。AIGC 创作、ストーリーボード作成、動画生成ワークフローの構築に最適です。

## プロジェクト概要

このプロジェクトは AI コンテンツ制作シーン向けに設計されており、視覚的なノードキャンバスを提供します。ユーザーはテキスト、画像、動画などのノードを接続して完全な生成パイプラインを形成できます。また、様々なモデルと API チャネルの設定をサポートしており、必要に応じて異なるプロバイダーと機能を切り替えて、異なる创作ニーズに対応できます。

## 主な機能

- ノードベースのキャンバスワークフロー
- テキスト生成
- 画像生成
- 動画生成
- 参照画像入力とマルチ画像連携
- 上流・下流ノードのコンテンツ受け渡し
- 生成結果のプレビューと保存
- マルチモデル / マルチチャネル切り替え
- デスクトップパッケージングとリリース（Windows など）
- タスク状況と生成進捗の表示

## 技術スタック

- **フロントエンドフレームワーク**: Vue 3
- **言語**: TypeScript
- **ビルドツール**: Vite
- **デスクトップ**: Electron
- **状態管理**: Pinia
- **ルーティング**: Vue Router
- **国際化**: vue-i18n
- **UI コンポーネント**: Element Plus
- **キャンバス/ワークフローオーケストレーション**: Vue Flow
- **3D/可視化**: Three.js / TresJS
- **テスト**: Playwright
- **パッケージツール**: electron-builder

## リポジトリ構造

- `src/`: コアビジネスソースコード
- `public/`: 静的リソースとアイコン
- `flow2api-main/`: 関連サービス/統合コンテンツ
- `scripts/`: ビルドとクリーンアップスクリプト
- `release/`: パッケージ出力ディレクトリ

## 環境要件

- Node.js 18+
- npm 9+
- Windows / macOS / Linux（パッケージ設定に依存）

## 依存関係のインストール

```bash
npm install
```

## ローカル開発

```bash
npm run dev
```

同時に Electron 関連のプロセスを実行するには、プロジェクトスクリプトに従って実行します：

```bash
npm run electron:dev
```

## プロダクションビルド

```bash
npm run build
```

## フロントエンドのみビルド

```bash
npm run build:vite
```

## デスクトップアプリのパッケージ

```bash
npm run electron:build
```

Windows インストーラーは `release/` ディレクトリに出力されます。

## よく使うスクリプト

- `npm run dev`: フロントエンド開発環境を起動
- `npm run build`: 完全なビルドとパッケージ
- `npm run build:vite`: フロントエンドビルドと型チェックのみ実行
- `npm run preview`: ビルド結果をプレビュー
- `npm run lint`: コードチェックと自動修正
- `npm run format`: `src/` をフォーマット
- `npm run test`: Playwright テストを実行
- `npm run clean:tmp`: 一時ファイルをクリーンアップ

## 設定方法

プロジェクトは設定画面で異なる API チャネルとモデルを設定し、テキスト、画像、動画生成に使用できます。

通常、以下を設定する必要があります：
- API アドレス / Base URL
- API キー
- 対象モデル
- プロバイダーグループ（優尚 / Flow2 など）

> 注意：実際のキーは GitHub に送信しないでください。ローカル設定または `.env` / 設定画面で保存してください。

## 機能のハイライト

- 視覚的なノードオーケストレーション、コンテンツワークフローの構築に適しています
- 複数の参照画像と上流・下流のコンテキスト注入をサポート
- 動画生成ノードは異なるモデルとパラメータ設定をサポート
- 生成タスクの状態管理と結果の書き戻しをサポート
- AIGC プロトタイプ検証、クリエイティブ制作、自動コンテンツ生成に適しています

## 注意事項

- `node_modules/`、`dist/`、`release/` などの成果物はリポジトリに送信しないでください
- 実際の API キーとアカウント情報を公開しないでください
- 現在のチャネルで一部のモデルが利用できない場合は、サポートされているモデルまたはチャネルに切り替えてください
